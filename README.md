## 🛠 Project Overview

This cheatsheet provides a quick visual reference for common HTML elements and CSS properties.  
It focuses on three main areas:

- HTML & CSS fundamentals
- The CSS Box Model
- Positioning and Display properties

The goal of the project is to create a **clean and readable reference page for beginners learning web development**.
